# O2 开发工具

见 [Github 页面](https://moushudyx.github.io/o2-dev-tools/)

## 参与贡献

1. 切出（fork）仓库到你的名下
2. 新建 feat-xxx / bugfix-xxx 分支
    1. `feat` 类型的分支一般用于添加**影响较大的功能**或**影响较大的改动**
    2. `bugfix` 类型的分支一般用于修复代码漏洞（可以简写为 `bug-xxx`）
3. 提交代码，注意提交的时候附带详细的说明
4. 新建合并请求
